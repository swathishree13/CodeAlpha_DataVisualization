# 📊 Employee Salary Analysis Dashboard

## Project Overview

This project analyzes employee salary data and transforms raw information into meaningful visualizations using Python, Matplotlib, and Seaborn. The dashboard highlights salary trends, departmental performance, and relationships between employee attributes.

## Objectives

* Analyze employee compensation patterns.
* Visualize salary distribution across departments.
* Identify relationships between experience and salary.
* Explore workforce demographics and performance.

# 📊 CodeAlpha - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Science Internship Program**. The objective of this project is to perform Exploratory Data Analysis (EDA) on a custom dataset containing inspirational quotes and authors to uncover patterns, trends, and insights using Python.

Exploratory Data Analysis is a crucial step in the data science workflow that helps understand the structure of data, identify anomalies, test assumptions, and prepare data for further analysis.

---

## 🎯 Objectives

* Understand the structure and characteristics of the dataset.
* Identify trends, patterns, and anomalies.
* Perform statistical analysis on the data.
* Visualize the data using charts and graphs.
* Detect potential data quality issues.

---

## 📂 Dataset Description

The dataset contains the following attributes:

| Column Name  | Description                        |
| ------------ | ---------------------------------- |
| Quote        | Text of the inspirational quote    |
| Author       | Name of the author                 |
| Quote_Length | Number of characters in each quote |

---

## ❓ Questions Asked During Analysis

1. How many quotes are present in the dataset?
2. How many unique authors contributed quotes?
3. Which author has the highest number of quotes?
4. Are there any missing values?
5. Are there any duplicate records?
6. What is the average length of a quote?
7. Which quote is the longest?
8. How are quote lengths distributed?
